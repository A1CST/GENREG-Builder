Radial Space

A fixed dot-lattice spun 1°/step. Each dot is an oscillator r·cos(T+φ₀). A signal is "addressed" to the dot it matches — 3 floats instead of the raw samples.

Fourier rebuilds your signal from endless waves that run the whole width. A short event forces it to stack up many of them. The radial version uses blips — little packets you drop right where the action is. Fewer pieces = smaller file + you know when things happen.
