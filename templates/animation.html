A VERY simple evolutionary model — mutation only, no crossover. Each genome is frame(4096) → tanh(encoder) → tanh(hidden) → shape logits. One clip per generation: the population sees a single clip's 24 frames (in temporal order), clip order shuffled each epoch — never all 240 at once. Fitness = accuracy on just that clip. Deliberately basic; a plateau is an expected, honest result.
