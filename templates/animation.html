The temporal radial stack — naming the animation from its motion

The animations below are the dataset. The model watches a 6-frame window of one of them and must say which animation it is — but the shape riding the path is randomized, the window starts anywhere in the clip, and the whole path is shifted by a random offset. Shape is a decoy and position is decorrelated: only the motion between frames names the animation (10 classes, chance 0.10). A twin model trains on the same sequences with the labels flipped — name the shape, with the motion as the decoy — so the two stacks show where evolution puts the work when the answer lives in a frame versus between frames. The current best radial setup (universal grammar, emergent-cap stacked spaces, no gradients) is wired to both below; results update live from the run exports.

The wiring
frames

Every frame is a row

The environment holds all 6 frames of every sequence as ordinary images (patch-PCA maps, label-free). Nothing temporal is designed in — the frames are just more data at this stage.

R0

Per-frame perception

Grammar-v2 spatial genomes evolve as usual; a genome's fitness column is its per-frame scalar averaged over the sequence. R0 becomes shape / position detectors — it cannot see motion.

hand-off

Time becomes channels

Each frozen R0 genome emits its spatial grid map for every frame; the R1 bank is laid out (genome × frame). The universal grammar's folds, shifts and moment stats now compose across time for free: the centroid's trajectory across frames is the path.

R1+

Temporal composition

Standard emergent-cap stacked spaces over the temporal bank — depth, size and primitive usage decided by the energy economy, not by hand. Closed-form head; test touched once.

What a temporal genome can say
# expressible in the universal grammar over the (genome × frame) bank:
falling  = comy-pool( shape-detector @ frame5 ) − comy-pool( shape-detector @ frame0 )
turning  = |com( detector @ f4 ) − com( shift(detector @ f2, dy,dx) )|
# zigzag reverses comy, loop bends comx AND comy, line holds comy flat —
# the same primitives that composed space, composing time

Same config, same sequences, one label flipped: the motion stack grew 5 spaces with the temporal space largest; the shape stack halted itself at 2 when depth stopped paying (R1: one genome, +0.0000). 4 · Gradient-free by construction

Feature genomes are evolved (tournament + crossover + energy economy); the only fitting anywhere is a closed-form ridge head. Inspect real genomes below — they are small trees of gates, folds and moment stats, not weight matrices. There is no backprop in the codebase path that produced these models.
