How a radial model trains (the v3 pipeline)

No gradients anywhere. The classifier is always closed-form; evolution builds only the features, and every structural property of a feature is a gene. The numbers below are live from the current run exports. The earlier rotation-cube demo is superseded by this page; the space-cousin finder sub-page is unchanged.

The pipeline — one feature genome's life
environment

Patch statistics

Label-free PCA maps of image patches, built once from data. Patch scale (4–14 px) is a gene — maps are built lazily for whatever scales evolution visits.

genome

A feature program

2–3 terms (each: a source channel bent through 1–2 evolved activations), folded by an op (×, min, |−|), pooled by an evolved soft window (center + width genes), optionally multiplied by a gate σ(k·g(x)) — conditional routing. Term sources can be raw maps or other genomes' outputs (meta).

fitness

Residual gain, honestly

Score = how much the feature improves the frozen ensemble's answers on a held-out val split (10k, rotated per stage). Computed closed-form via a rank-1 Schur border — one Cholesky per round, the whole population in fused matmuls. Test is touched once, at the end.

selection

Tournament + crossover + energy

Tournament-of-3 parents, uniform per-gene crossover (p=0.5), self-adapted mutation scale. Downstream stages run the energy economy: existing costs, outputting costs, only above-median contribution earns — the starved die and their slots go to children.

freeze

Freeze & compose

Each round the best ≤8 decorrelated survivors are frozen into the model forever. Rounds repeat until several go dry. Stacks: a converged model's outputs become the next stage's data.

scale

Seed farming

Independent runs (different seeds, different machines) discover complementary features. Their union with one head refit beats any single tower — the diversity curve below.

Why unions win: every seed converges to a different ~500-genome vocabulary; the features are complementary, so one closed-form head over the union is worth several points over any single run — at the cost of a few seconds of refit. Model size stays tiny: a genome is ~15 numbers.
Ground rules that make the numbers honest: fitness never sees test; each stage selects against a fresh val window; every claimed number is a single test measurement of one final model.
