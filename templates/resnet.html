ResNet — gradient-free

The whole lab is gradient-free (rule #1: no gradients, no backprop, no hybrid). This page asks the ResNet question inside that law: can evolution discover the residual skip and stack it usefully, scored only by a closed-form linear read-out on a held-back val split? Same machinery as Radial v2 — swapped grammar.

The residual genome
stem
2–4 label-free patch-PCA channel maps (the environment). Channel count fixed through the stack so the identity add stays valid.
residual blocks — depth is a gene
h ← h + gain·act(a·(mix·h)+b). The skip is the ResNet gene; mix is the 1×1-conv analog; act is the 8-function catalog. New blocks bootstrap as near-no-op then evolve.
head
weighted channel collapse → one map → evolved soft spatial window (cx, cy, σ) → one scalar per image via mean / max / std.
Read-out

Each round breeds a population (comma-GA, tournament + elitism), freezes the top decorrelated survivors as columns, and reads them out with a closed-form ridge. Test touched exactly once. No gradients anywhere.

Pipeline: resnet_evo.py. Artifacts (checkpoint + result JSON) write to F:\Resnet, not C:. Full run: python resnet_evo.py. Smoke: python resnet_evo.py --smoke. Runs appear on Runs under resnet_evo.
One residual network → one scalar feature. Evolution is the only optimizer — it sets scale, channels, depth, mixing, activations, residual gains, and the read-out window. The bank of frozen residual features is scored by a linear head that never sees a gradient.
